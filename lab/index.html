  • マーカーの中心：position="0 0 0"
  • マーカーの1辺：scale="1 1 1"（長方形マーカーの場合は？）
  • 座標系：右手系
    A-Frame：X = 右 / Y = 上 / Z = 手前
    AR.js：X = 右 / Y = 手前 / Z = 下
  • オブジェクト座標指定：オブジェクトの中心（内部）
  • 画像：rotation="0 0 0"
    ⇒　XY平面上にZ正方向を正面（画面下向き）に貼り付け
    ⇒　画面正面方向への配置 rotation="-90 0 0"（回転方向：右手系）
    ⇒　回転の処理順 = X→Y→Z（たぶん）
  • 作業順：rotation → position
  • 秒間6～7fr前後？（js.setInterval(,1)でも、秒間実行6～7回のようだ）
  • HTML記載順：前＝前レイヤー／後＝後レイヤー
    前レイヤーの透過部分＝背景の画像に対しalpha値を上書きし、background=a-skyを表示
  • 記述方法に互換性がなくとも、多少頑張って最新版にしないと、色々不都合があるかも。
    （例）ジャイロ酔い＝ブレブレ改善
